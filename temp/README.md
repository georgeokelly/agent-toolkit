# temp

Temporary directory for generated artifacts used to verify that rules in this repo work as expected.

本目录用于存放临时生成内容，主要用于验证本仓库 rules 功能是否按预期工作。

## Applicable Content / 适用内容

- Temporary files generated by scripts or agents / 脚本或 agent 运行时生成的临时文件
- Intermediate artifacts and sample I/O for functional verification / 用于功能验证的中间产物与样例输入输出
- One-off debug logs (safe to clean up anytime) / 一次性调试记录（可随时清理）

## Conventions / 使用约定

- Contents here are ephemeral — naming and structure are not guaranteed stable / 本目录内容默认不作为长期资产，不保证命名和结构稳定
- Clean up promptly after verification to avoid repo noise / 验证完成后应及时清理，避免仓库噪音
- Results worth keeping long-term should be moved to a proper directory with documentation / 需要长期保留的结果应迁移到正式目录并补充说明

## Git Note / Git 说明

- Keep ignored during normal development to reduce noise in commits / 日常开发可保持 ignore 状态，降低无关变更进入提交的概率
